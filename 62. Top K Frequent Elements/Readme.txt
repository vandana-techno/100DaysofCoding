Question link - https://leetcode.com/problems/top-k-frequent-elements/description/

Question - Given an integer array nums and an integer k, return the k most frequent elements. You may return the answer in any order.

 

Testcase 1:

Input: nums = [1,1,1,2,2,3], k = 2

Output: [1,2]



Testcase 2:

Input: nums = [1], k = 1

Output: [1]

 

Constraints:

1. 1 <= nums.length <= 105

2. -104 <= nums[i] <= 104

3. k is in the range [1, the number of unique elements in the array].

4. It is guaranteed that the answer is unique.


Language used- Java

Platform- Leetcode
